## Nuestra aplicación tiene la siguientes funcionalidades: 
1. Crear grupo
2. Unirse grupo
3. Inicar chats
    
    3.1. Enviar mensaje de texto

    3.2. Enviar mensaje de audio

    3.3. Ver historial
